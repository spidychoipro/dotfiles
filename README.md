## WSL Setup

WSL Arch Linux 셋업 스크립트는 `wsl/setup.sh`에 있음

실행 방법:
```bash
bash setup.sh
```

설치되는 것들:
- zsh + zsh-autosuggestions + zsh-syntax-highlighting (Dracula theme)
- neovim, python, nodejs
- 한국어 UTF-8 로케일
